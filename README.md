# Py .min.css 

## Description

CSS minimize tools there are many in Internet, but it is offline CSS minimize. 

## Uses

#### Minimize:

To minimize a CSS file, get the `path` of the file and open a command line:
```
    python css-min.py -m=path
```
If the program runs without problem must print 
>Done!
>File Name: `path` ending with `.min.css`

#### Help:

To see the help of the program run:
```
    python css-min.py -h
```
